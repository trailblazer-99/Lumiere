Installation Instructions (One-time Setup)
1

Download and Trust the Security Certificate

Because Lumière is sideloaded, Windows requires trusting its developer certificate before installation. Click Download Certificate above to download LumiereMediaPlayer.cer, then:

  1. Double-click the downloaded LumiereMediaPlayer.cer file.
  2. In the Dialog box that opens, click the Install Certificate... button.
  3. Select Local Machine (requires administrator access) and click Next.
  4. Choose Place all certificates in the following store and click Browse....
  5. Select the Trusted People store and click OK.
  6. Click Next, then Finish. You should see a confirmation saying "The import was successful."
2

Run the App Installer

Click the Install Lumière Player button to download and run the .appinstaller file. Windows App Installer will launch automatically to download the correct bundle for your system (x64 or ARM64) and install it.

Note: If Windows complains about the file or says it cannot open the appinstaller, double-click the raw .msixbundle file downloaded via the backup link to install it directly.

3

Automated Background Updates

Once installed using the App Installer, Windows will automatically query the repository's portal on GitHub Pages for updates. Every time changes are pushed to the repository, the app will be built, signed, and updated on your system seamlessly in the background!
